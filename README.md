- CADASTRO DE TIPOS DE ENTRADA
- CADASTRO DE TIPOS DE SAIDA

  -  tipo_entrada, descrição, valor, data_entrada
- SAIDAS

- RELATORIOS (mensal, quinzenal, periodos …)
  - DESPESAS POR TIPOS
  - RECEITAS POR TIPOS
- CARTEIRA
